# Bike Sharing Analysis Dashboard

## Setup Environment
- **Create a folder for the project**
- **Install the required libraries:**
  ```bash
  pip install pandas matplotlib seaborn streamlit
  ```

## Run Streamlit App
- **Ensure you are in the project directory**
- **Execute the following command:**
  ```bash
  streamlit run dashboard/dashboard.py
  ```
  *(Note: If the command above fails, use `python -m streamlit run dashboard/dashboard.py`)*

## Project Structure
- `dashboard/`: Contains `dashboard.py` and `main_data.csv`.
- `data/`: Contains original datasets (`day.csv`, `hour.csv`).
- `Proyek_Analisis_Data.ipynb`: Data analysis and exploration notebook.
- `requirements.txt`: List of dependencies.
- `README.md`: Project documentation.
